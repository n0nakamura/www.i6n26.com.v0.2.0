== 辞書に関してやりたいこと

私は辞書に次の機能を追加したいと考えている。

* ある単語を検索した回数の記録。
* ある単語へのコメントの追加。
* ウェブページ上の語句を辞書で検索したとき、このウェブページのURIを該当語句と紐づける。
* IMEの辞書と相互運用。
