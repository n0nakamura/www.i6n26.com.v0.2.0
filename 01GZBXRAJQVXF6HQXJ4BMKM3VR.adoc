== 総称について

公な文書を作成するとき、補聴器、人工内耳、骨伝導補聴器などの総称名を必要とする場面があります．私はこれまで、厚生労働省のウェブページで採用されている「補装具」に「聴覚」という修飾子を付与した、「聴覚補装具」という自作の総称名を用いてきました．最近、総称名について考える機会があったため、改めて総称名について考えたいと思います．
